# `compressor`


## `ratio`

  In format of _signal:attenuation_ where `signal` has `attenuation` upon crossing `threshold`


### _(mild)_ 
  - `2:1` Drums


### _(moderate)_ 
  - `3:1` Bass / Mixbus
  - `4:1` ...


### _(medium)_ 
  - ...


### _(strong)_ 
  `10:1` _(limiter)_ Drum Room
  `20:1` _(limiter)_ See Limiter
